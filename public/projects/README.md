# Project assets

Each app has its own folder. Paste files here — no code changes needed for preview screens if you use the naming below.

## Folder structure

```
public/projects/
  evolv-solutions/
    icon.png          ← app icon (already set)
    screens/          ← paste preview screenshots here
      01.png
      02.png
      03.png
  indopin/
    icon.png
    screens/
  raddiwalla/
  daylight/
  interpretrain/
  digicat-jewelry/
  jewelry-erp/
  testpaper-live/
```

## Preview screens

1. Open the project folder (e.g. `evolv-solutions/screens/`)
2. Add images named **`01.png`**, **`02.png`**, **`03.png`**, … (up to `20.png`)
3. Supported: `.png`, `.jpg`, `.jpeg`, `.webp`
4. They appear automatically on the project detail page

You can also list custom paths in `src/data.json` under each project's `"screens"` array.
